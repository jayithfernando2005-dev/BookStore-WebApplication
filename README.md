# 📚 BookStore Web Application

A modern web-based bookstore application developed using HTML, CSS, JavaScript, PHP, and MySQL. The system allows users to browse books, explore categories, view book details, and enjoy a responsive online bookstore experience.

---

## 🚀 Features

- 📖 Browse available books
- 🔍 Search and explore book collections
- 📂 Category-based book organization
- 🛒 User-friendly bookstore interface
- 📱 Responsive design for different screen sizes
- 🎨 Modern UI using HTML, CSS, and JavaScript
- 🗄️ Database integration with MySQL
- ⚡ Backend functionality powered by PHP

## ⚙️ Installation Guide

### Prerequisites

Before running the project, ensure you have:

- WAMP Server installed
- PHP installed
- MySQL installed
- Web Browser
- Visual Studio Code (Optional)

### Steps

1. Clone the repository

```bash
git clone https://github.com/jayithfernando2005-dev/BookStore-WebApplication.git
```

2. Move the project folder into your WAMP directory

```text
wamp64/www/
```

3. Start WAMP Server

4. Import the database into phpMyAdmin

5. Open your browser and navigate to:

```text
http://localhost/BookStore-WebApplication
```
